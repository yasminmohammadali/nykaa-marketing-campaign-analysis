<h1>📊 Nykaa Marketing Campaign Analysis Dashboard (Excel & Tableau)</h1>

<p>
This project presents a comprehensive marketing campaign analysis for 
<strong>Nykaa</strong>, built using the same dataset across 
<strong>Microsoft Excel</strong> and <strong>Tableau</strong>.
It highlights campaign performance, customer segmentation, ROI, and channel efficiency.
</p>

<hr>

<h2>📁 Project Overview</h2>

<p>
The goal of this project is to analyze marketing campaign data and extract insights related to:
</p>

<ul>
  <li>Channel performance (Email, Facebook, Google, Instagram, WhatsApp, YouTube)</li>
  <li>Customer segmentation</li>
  <li>Conversion rates and ROI</li>
  <li>Acquisition costs and CPC</li>
  <li>Revenue trends over time</li>
</ul>

<hr>

<h2>🛠 Tools Used</h2>

<ul>
  <li><strong>Microsoft Excel</strong> – Data cleaning, KPI calculation, and dashboard creation</li>
  <li><strong>Tableau</strong> – Interactive dashboard and visual storytelling</li>
</ul>

<hr>

<h2>📊 Dashboards</h2>

<h3>1️⃣ Excel Dashboard</h3>

<p>

<p><strong>Key Insights:</strong></p>

<ul>
  <li>Instagram and YouTube show strong ROI performance.</li>
  <li>Google has the highest CPC, indicating higher competition.</li>
  <li>Conversions are fairly balanced across all channels.</li>
  <li>A noticeable dip in revenue occurs mid-year, followed by recovery.</li>
</ul>

<hr>

<h2>📈 Key Metrics Explained</h2>

<ul>
  <li><strong>ROI (Return on Investment):</strong> Revenue generated per unit of cost</li>
  <li><strong>CPC (Cost Per Click):</strong> Average cost paid per click</li>
  <li><strong>Conversion Rate:</strong> Percentage of users converted into leads</li>
  <li><strong>Acquisition Cost:</strong> Cost incurred to acquire a customer</li>
</ul>

<hr>

<h2>🔍 Insights & Learnings</h2>

<ul>
  <li>High ROI channels should receive increased budget allocation.</li>
  <li>High CPC channels need optimization to reduce costs.</li>
  <li>Customer segmentation helps target high-value groups effectively.</li>
  <li>Seasonal trends impact revenue and campaign performance.</li>
</ul>
